SkypeMessagePrefixer for Skype for Windows
==
This applications runs in the background and prefixes the Skype chat input with a double exclamation mark, thus making all messages monospace and disabling emojis. The application does not add exclamation marks when the message starts with a URL.

Author: Max Vollmer

I was lazy, there is no GUI and the only way to terminate the application is by killing it with the Windows task manager.
